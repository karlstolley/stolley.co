Course Development: Information Technology and Management

  • ITMD 469/545: Web Real-Time Communications

    Completely original syllabus providing coverage of the W3C WebRTC 1.0: Real-time Communication Between Browsers specification as well as other extant real-time/streaming web APIs and libraries, including WebSockets and Web Notifications. (2 sections taught)

    • Syllabus: https://courses.stolley.co/rtc/

  • ITMD 467/567: Web Systems Integration

    Completely original syllabus created to distinguish the course from other standalone web-application courses offered by the department and include coverage of API production and consumption as part of a service-oriented application architecture. Broad coverage of deployment and monitoring as well as automated testing and test-driven development. (3 sections taught)

    • Syllabus: https://courses.stolley.co/wsi/

  • ITMD 469/569: Data APIs

    Completely original course and syllabus created to provide students with in-depth experience working with web-available data APIs. Includes substantial instruction in REST architecture and API design, as well as classic (CSV, XML, SOAP) and modern (JSON, BSON, YAML) data-serialization formats. Coverage of server-side JavaScript in Node.js, one of only a handful of ITM courses to do so. (1 section taught)

    • Syllabus: https://courses.stolley.co/summer-apis/

  • ITMD 413/513: Open-Source Programming: Ruby on Rails

    Completely original course and syllabus created to provide students with a strong conceptual foundation of MVC architecture on the web as expressed in Ruby on Rails. Includes coverage of Ruby syntax, structures, and custom, and the culture and function of open-source programming. (1 section taught)

    • Syllabus: https://courses.stolley.co/summer-rails/

  • ITMD 361: Fundamentals of Web Development

    Completely original course syllabus written to introduce students to web development according to contemporary web standards and best practices (e.g., mobile-first responsive web design; progressive enhancement). Includes substantial instruction in the foundations of programming generally and developer practice, including Git-based workflows. (numerous sections taught)

    • Adopted as the standard department syllabus for all ITM faculty.
    • Syllabus: https://courses.stolley.co/fwd/

  • ITMD 362: Human-Computer Interaction and Web Design

    Completely original course syllabus written to provide solid foundations in human-computer interaction, including user testing and user-centered interface design. Coverage of modern CSS layout modules including CSS Flexible Boxes (Flexbox) and CSS Grid. Includes substantial instruction in theories of HCI as well as design topics including color theory, typography, and grid-based layout. (numerous sections taught)

    • Adopted as the standard department syllabus for all ITM faculty.
    • Subject to departmental assessment and ABET accreditation in 2017 and 2018; received a 3.92 score out of 5 in 2017; jumped to a 4.61 score out of 5 in 2018. Also reflected department-leading scores on ABET outcomes both years.
    • Syllabus: https://courses.stolley.co/hci/
